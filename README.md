## 🚀 Οδηγίες Εκτέλεσης με Docker

1. Τοποθετήστε τα παρακάτω αρχεία στον ίδιο φάκελο:
   - `app.py`
   - `sky.jpg`
   - `requirements.txt`
   - `Dockerfile`

2. Ανοίξτε τερματικό (terminal) στον φάκελο και δώστε την εντολή:

```bash
docker build -t molebio-app .
```

3. Μόλις ολοκληρωθεί το build, εκτελέστε:

```bash
docker run -p 8501:8501 molebio-app
```

4. Ανοίξτε τον browser και μεταβείτε στη διεύθυνση:

```
http://localhost:8501
```
